Sequence

A sequence is an ordered list of numbers (e.g., an); the numbers are called "elements" or "terms". Every convergent sequence is bounded, thus an unbounded sequence is divergent.

$\lim\limits_{n \to \infty} a_n = \lim\limits_{n \to \infty} c_n = L$ then $\lim\limits_{n \to \infty} b_n = L$ $\lim\limits_{n \to \infty} a_n = L$

an ≤ bn ≤ cn

Series

A series is the sum of the terms of a sequence: $\sum\limits_{n=1}^\infty a_n$.

Series Test Converge Diverge Notes

Divergence

 N/A $\lim\limits_{n\to\infty} a_n \ne 0$

Doesn't show convergence and the converse is not true

Integral

 if $\int\limits_1^\infty f(x) dx$ converges if $\int\limits_1^\infty f(x) dx$ diverges

f(x) must be positive, decreasing, and continous, also f(n) = an for all n

Root

 $\lim\limits_{n\to\infty}\sqrt[n]{|a_n|} = L < 1$ $\lim\limits_{n\to\infty}\sqrt[n]{|a_n|} = L > 1 \text{ or } \infty$

inconclusive if L = 1

Ratio

 $\lim\limits_{n\to\infty} \left| \frac{a_{n+1}}{a_n}\right| = L < 1$ $\lim\limits_{n\to\infty} \left| \frac{a_{n+1}}{a_n}\right| = L > 1 \text{ or } \infty$

inconclusive if L = 1

Direct Comparison

 0 ≤ an ≤ bn for all n and $\sum\limits_{n=1}^{\infty} b_n$ converges 0 ≤ bn ≤ an for all n and $\sum\limits_{n=1}^{\infty} b_n$ diverges

an, bn > 0

Limit Comparison Absolute Conditional

$\lim\limits_{n\to\infty} \frac{a_n}{b_n} = L$ and $\sum\limits_{n=1}^{\infty} b_n$ converges $\sum\limits_{n=1}^{\infty} | a_n | = 0$ $\sum\limits_{n=1}^{\infty} | a_n |$ diverges but $\sum\limits_{n=1}^{\infty} a_n$ converges

$\lim\limits_{n\to\infty} \frac{a_n}{b_n} = L$ and $\sum\limits_{n=1}^{\infty} b_n$ diverges

an, bn > 0 and L is a positive constant, if L is ∞ or 0, then pick a different bn Definition of absolutely convergent, the sum is independent of the order in which the terms are summed The sum is dependent of the order in which the terms are summed

Alternating

 $\sum\limits_{n=1}^\infty (-1)^{n-1} a_n$ 0 < an + 1 ≤ an for all n and $\lim\limits_{n \to \infty} a_n = 0$ N/A

Geometric

 $\sum\limits_{n=1}^\infty ar^{n-1}$ |r| < 1 and converges to $\frac{a}{1-r}$ |r| ≥ 1 finite sum of the first n terms: $= \frac{a(1-r^n)}{1-r}$

P-Series Power

$\sum\limits_{n=1}^\infty \frac{1}{n^p}$ $\sum\limits_{n=0}^\infty c_n (x-a)^n$

p > 1 $\begin{array}{l} i, \text{converge if } x=a \\ ii, \text{converge for all } x \\ iii, \text{converge if } |x-a|<R \end{array}$

p ≤ 1

cannot calculate sum R is the radius of convergence, you need to check the end points for convergence too. Typically use Ratio Test.

Taylor

 $\sum\limits_{n=0}^\infty \frac{f^n (a)}{n!} (x-a)^n$ |x − a| < R |x − a| > R Taylor series is centered about a. Same note as power series

Maclaurin

 $\sum\limits_{n=0}^\infty \frac{f^n (0)}{n!} (x)^n$ |x| < R |x| > R A Macluarin series is a Taylor series centered about 0. Same note as power series
